# Simple Interest Calculator

This project is a simple interest calculator implemented using a Bash script.

## Formula
Simple Interest = (Principal × Rate × Time) / 100

## Description
The script takes three inputs from the user:
- Principal amount
- Rate of interest
- Time period

It then calculates and displays the simple interest.

## Example
If Principal = 1000, Rate = 5%, Time = 2 years  
Simple Interest = (1000 × 5 × 2) / 100 = 100

## Usage
Run the script using:
bash simple-interest.sh
